Detailed Description

Random generator for uniform distributions.

Remarks
they try to have only one (common) interface
  • random returns random numbers
    random(state, r)
    • state is a random_state type variable
    • r is either a real scalar or a 1D, 2D or 3D array
  • seed Seed the generator
    seed(key, state)
    • key is, if present, either an integer scalar or 1D array
    • state is a random_state type variable
